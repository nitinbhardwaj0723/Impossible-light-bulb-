# 💡 Impossible Light Bulb

A fun and interactive project that demonstrates a simple yet clever concept —
whenever you try to turn the light ON, opening the door automatically turns it OFF 😄

This creates an "impossible" situation where the light cannot stay ON.

---

## 🚀 Features

* 🔌 Light ON/OFF control
* 🚪 Door-based trigger mechanism
* ⚡ Automatic light OFF when door opens
* 🎯 Simple logic-based project
* 🛠️ Easy to understand and implement
